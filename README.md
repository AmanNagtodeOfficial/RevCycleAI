# RevCycle AI

Next-gen AI-powered Revenue Cycle Management (RCM) platform built with Next.js, Firebase, and Genkit.

## 🚀 Final Step: Push to your GitHub

To finalize the sync to your private GitHub repository, please run these commands in your **local terminal** (inside this project folder):

### 1. Initialize and Commit
```bash
git init
git add .
git commit -m "Initial commit: Professional RCM Platform"
git branch -M main
```

### 2. Connect to GitHub
Go to [github.com/new](https://github.com/new), create a **Private** repository named `revcycle-ai`, then run:
*(Replace `<YOUR_USERNAME>` with your actual GitHub username)*

```bash
git remote add origin https://github.com/<YOUR_USERNAME>/revcycle-ai.git
```

### 3. Push the code
```bash
git push -u origin main
```

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Database**: Firebase Firestore
- **Authentication**: Firebase Auth
- **AI**: Genkit (Google Gemini)
- **UI**: ShadCN UI + Tailwind CSS
- **Icons**: Lucide React
